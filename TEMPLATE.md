# Day Template

Use this template when creating new day folders. Each day should have:

```
dayXX-slug-name/
├── README.md          # Concept explanation (what you'll learn, why it matters)
├── prompt.md          # The prompt to paste into your coding assistant
├── requirements.txt   # Python dependencies for this day
└── main.py            # Reference implementation (generated from the prompt)
```

## README.md Structure

1. **The Concept** — 2-3 paragraphs explaining what we're learning and why
2. **Setup** — Any prerequisites or installations needed
3. **What You'll Build** — Brief description of the hands-on project
4. **Key Takeaways** — 3-5 bullet points summarizing what was learned

## prompt.md Structure

1. Clear description of what to build
2. Numbered requirements (what the script should do)
3. Specific models/libraries to use
4. Educational comments requirement
